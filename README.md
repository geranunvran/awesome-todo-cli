# Awesome ToDo CLI 📝

A lightweight, no-nonsense command-line to-do list app.  
Manage your tasks directly from your terminal like a pro.

## Features
- ✅ Add, list, and remove tasks
- 📅 Deadlines and priorities
- 💾 Saves to local JSON file
- 🧪 Simple test suite
